DESCRIÇÃO DO PROJETO:
SISTEMA GoumertOS


Objetivo do projeto:
O Sistema de Gerenciamento de Restaurante (GourmetOS) tem como objetivo organizar e otimizar os processos internos de um restaurante, facilitando o controle de pedidos, mesas, cardápio e pagamentos.
O sistema busca melhorar a experiência do cliente, reduzir erros operacionais e aumentar a eficiência da equipe, permitindo um atendimento mais rápido e organizado.


Tecnologias:
Front-end:React (Java Script) , TailwindCss , HTML
Back-end : Laravel (PHP)
Banco de dados : MySQL
Controle de versão: Git
Metodologia : Organização baseada em boas práticas de desenvolvimento e padrões de projeto

Como rodar o projeto:
Atualmente, o projeto GourmetOS está sendo executado em um servidor local utilizando o XAMPP, que permite a hospedagem da aplicação em ambiente de desenvolvimento, com suporte ao PHP e ao MySQL por meio dos serviços Apache e MySQL. Esse ambiente possibilita a realização de testes, ajustes e validações do sistema de forma segura e controlada durante a fase de desenvolvimento. Posteriormente, o projeto será publicado em um servidor gratuito na internet, tornando-se acessível online e disponível para utilização remota.


1-Instalar o XAMPP e iniciar os serviços Apache e MySQL.
2-Clonar ou copiar o projeto para a pasta htdocs do XAMPP.
3-Criar o banco de dados no MySQL e importar o arquivo .sql do projeto (se disponível).
4-Configurar as credenciais de conexão com o banco no arquivo de configuração do PHP.
5-Iniciar o front-end com React utilizando:
npm install
npm start
6-Acessar o sistema pelo navegador através do endereço:
http://localhost
